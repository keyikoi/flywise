### Slide 03 - 核心痛点
**标题：** 核心痛点
**副标题：** 自然语言的困境

**问题卡片：**
- UI 调整时，自然语言不容易传达精确意图

**例子 1（坏）：**
- 代码：`"把卡片间距调大一点"`
- 结果：→ AI 可能改错地方

**例子 2（坏）：**
- 代码：`"弹窗居中"`
- 结果：→ 可能只水平居中

---

### Slide 04 - 解决方案
**标题：** 解决方案
**副标题：** Figma MCP 联动

**方法卡片：**
- 直接给设计截图/节点，AI 看到 exact 样式

**案例 1（好）：**
- 代码：`gap-4 → gap-6`
- 描述：卡片间距调整

**案例 2（好）：**
- 代码：`top-4 → top-1/2 -translate-y-1/2`
- 描述：弹窗垂直居中

**优势：** 一次到位，避免多轮对话

### Slide 07 - 协作技巧 3
**标题：** 协作技巧 3
**副标题：** Git 版本管理

**卡片内容：**
- 标题：工作流
- 列表：
  - 每次成功改动就 commit
  - 错了随时回滚
  - commit message 作为"外部记忆"

**代码示例：**
```
git commit -m "feat: 添加滑动卡片动画"
git commit -m "fix: 修复卡片间距问题"
git commit -m "feat: 集成 Figma MCP"
```
