03

核心痛点

自然语言的困境

❌ 问题

UI 调整时，自然语言不容易传达精确意图

例子 1
"把卡片间距调大一点"

→ AI 可能改错地方
(margin? gap? 哪个组件？)

例子 2
"弹窗居中"

→ 可能只水平居中
忘了垂直居中

04

解决方案

Figma MCP 联动

✅ 方法

直接给设计截图/节点，AI 看到 exact 样式

案例 1
gap-4 → gap-6

卡片间距调整

案例 2
top-4 → top-1/2 -translate-y-1/2

弹窗垂直居中

🎯 优势：一次到位，避免多轮对话

06

AI 协作技巧 2

改 Bug 的正确姿势

📸 截图大法

  • 网页报错 → 截图保存
  • 丢给 AI 自己看错误信息
  • 比文字描述高效 10 倍

AI 看图能力 > 听你描述能力

07

AI 协作技巧 3

Git 版本管理

📝 工作流

  • 每次成功改动就 commit
  • 错了随时回滚
  • commit message 作为"外部记忆"
git commit -m "feat: 添加滑动卡片动画"
git commit -m "fix: 修复卡片间距问题"
git commit -m "feat: 集成 Figma MCP"

心得总结

🎨 设计部的优势

💡 核心心法

AI 是强大的，但需要明确的边界和持续的提醒

Vibe Coding = 人机协作，不是机器替代
